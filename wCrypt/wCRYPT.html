The KEY is the string based password. But as we know that a STRING cannot be used for calculation purposes and thus cannot help us in ENCRYPTION in any way, thus we have to somehow convert the string into INTEGER ARRAY and then it will be useful.

One of the simple ways of doing this is do LOOP over every CHARACTER in the STRING (key) and get its UNICODE INDEX. Arranging all the UNICODE INDEXs will result in a INTEGER based LIST which can now be used for calculation.

In PYTHON getting the UNICODE INDEX is very easy we can use the FUNCTION ord() for achieving this goal. The ord() FUNCTION takes in a CHARACTER and returns an INTEGER in the range 0-65535 (both included). Now we also know that every BYTE can hold a number from 0-255 (both included), thus we have to convert this ARRAY such that we get another ARRAY with the range 0-255 (both included).
